# App Icon

To generate app icons:

1. Place your 1024x1024 icon at `assets/icon/app_icon.png`
2. Place your foreground icon at `assets/icon/app_icon_foreground.png` (Android adaptive icon)
3. Run: `dart run flutter_launcher_icons`

Colors:
- Background: #16A34A (ReLoop Green)
- Platform: Android & iOS
